# Каталог товаров

Экзаменационный проект по дисциплине: Создание веб-приложений с использованием Angular и React.

## 📋 Описание:

Веб-приложение “Каталог товаров” , где можно просматривать данные, искать, фильтровать и добавлять новые элементы через интерфейс с использованием React.
Приложение демонстрирует работу с состояниями (State), побочными эффектами (Effects) и оптимизацией рендеринга (Memo).

## ✨ Функционал:

- Просмотр списка: Отображение товаров с изображениями, ценой и статусом.
- Поиск: Фильтрация товаров по названию в реальном времени.
- Расширенные фильтры:
- Фильтрация по категории.
- Фильтрация по статусу (В наличии / Нет в наличии).
- Кнопка сброса всех фильтров.
- Детальный просмотр: Блок с подробной информацией о выбранном товаре (таблица характеристик).
- Добавление товаров: Форма с валидацией для добавления новых позиций в каталог.
- Обработка ошибок и загрузки: Имитация запросов к серверу с индикатором загрузки и возможностью возникновения ошибки (с кнопкой повтора).
- Адаптивность изображений: Картинки автоматически вписываются в контейнер (object-fit: contain) с заглушками при ошибке загрузки.

## 🎨 Технологии:
- React 18: Хуки (useState, useEffect, useMemo).
- CSS-in-JS: Стилизация через inline-styles для компонентного подхода.
- Верстка: Flexbox для адаптивного интерфейса.


## 🛠 Установка и запуск

### Предварительные требования
- **Node.js >= 14.x**
- **npm >= 6.x или yarn >= 1.22.x**
- **Visual Studio Code** 

### Шаги установки

1. **Клонирование репозитория**
   ```bash
   git clone https://github.com/Azat-oss/product-catalog.git
   cd product-catalog
   ```
2. Восстановление зависимостей:
   ```bash
   npm install
   ```

3. Запуск приложения:
   ```bash
   npm start

Приложение будет доступно по адресу: http://localhost:3000

## 📝 Особенности реализации

### Имитация API
#### В App.js реализована функция fetchData, которая с задержкой в 1.2 секунды:

- В 60% случаев успешно загружает данные.
- В 40% случаев имитирует ошибку сервера, чтобы продемонстрировать работу компонента ErrorMessage.
#### Валидация формы
AddItemForm проверяет:

- Наличие названия и описания.
- Выбор категории.
- Корректность цены (положительное число).
